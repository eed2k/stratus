Weather Station Manager — Desktop Setup

Welcome to Stratus Desktop

Stratus is a professional weather station management application for Campbell Scientific and compatible data loggers. This desktop edition includes serial port connectivity for direct RS232/USB logger connections.

  • Real-time weather data monitoring & visualisation
  • Serial Monitor for RS232/USB data logger connections
  • Windy weather map integration
  • Lightning strike data (AfriGIS)
  • Automated data collection & reporting
  • Alarm management & notifications
