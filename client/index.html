Stratus Weather Monitoring Platform

Professional weather station management for Campbell Scientific dataloggers.

Features

  • Real-Time Weather Monitoring - Live data updates with WebSocket connections
  • Interactive Charts - Temperature, pressure, wind roses, and custom time ranges
  • Smart Alerts - Configure alarms for weather thresholds with email notifications
  • Report Generation - PDF and CSV reports with statistics and date ranges
  • Multi-Station Support - Manage multiple weather stations from one dashboard
  • Dropbox Sync - Automatic TOA5 data file synchronization from LoggerNet

Supported Equipment

Campbell Scientific CR300, CR1000X, CR6 dataloggers with TOA5, Array, and PakBus protocols.
