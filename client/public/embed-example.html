Stratus Weather Widget

Embed real-time weather data on any website

Live Widget Demo

HTML Usage

Add this code to your HTML page to embed the weather widget:

<!-- Add a container where you want the widget -->
<div 
  data-stratus-widget 
  data-station="1" 
  data-server="https://stratusweather1.dynv6.net"
  data-theme="light"
></div>

<!-- Include the widget script (at end of body) -->
<script src="https://stratusweather1.dynv6.net/api/embed/widget.js"></script>

Widget Options

Attribute Description Default
data-station Station ID to display Required
data-server Stratus server URL Current origin
data-theme Widget theme: "light" or "dark" light
data-show-rain Show rainfall data true
data-show-solar Show solar radiation data true

JavaScript API

For more control, initialize widgets programmatically:

// Manual initialization
StratusWidget.init(document.getElementById('my-widget'), {
  station: 1,
  server: 'https://stratusweather1.dynv6.net',
  theme: 'dark',
  showRain: true,
  showSolar: true,
  autoRefresh: true,
  refreshInterval: 300 // seconds
});

React Component Usage

import { useEffect, useRef } from 'react';

function StratusWeatherWidget({ stationId, server, theme = 'light' }) {
  const containerRef = useRef(null);

  useEffect(() => {
    // Load the widget script
    const script = document.createElement('script');
    script.src = \`\${server}/api/embed/widget.js\`;
    script.async = true;
    script.onload = () => {
      if (window.StratusWidget && containerRef.current) {
        window.StratusWidget.init(containerRef.current, {
          station: stationId,
          server: server,
          theme: theme
        });
      }
    };
    document.body.appendChild(script);

    return () => {
      document.body.removeChild(script);
    };
  }, [stationId, server, theme]);

  return <div ref={containerRef} />;
}

// Usage:
<StratusWeatherWidget 
  stationId={1} 
  server="https://stratusweather1.dynv6.net" 
  theme="dark" 
/>

REST API Endpoints

Access weather data directly via these public API endpoints:

Endpoint Description
GET /api/embed/stations List all available weather stations
GET /api/embed/station/:id Get station info and latest weather data
GET /api/embed/station/:id/data?hours=24&limit=500 Get historical weather data for charts
