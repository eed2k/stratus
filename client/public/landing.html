Stratus

Professional Weather Monitoring Platform

Stratus is a weather monitoring system designed for Campbell Scientific dataloggers. Monitor real-time weather data, generate reports, configure alarms, and manage multiple stations from one dashboard.

Features

Real-Time Monitoring

Live weather data updates via WebSocket connections. See temperature, humidity, wind, rainfall, and more as it happens.

Interactive Charts

Temperature, pressure, wind roses, and custom time range charts for visualizing weather trends.

Smart Alarms

Configure alarms for temperature, wind, and rainfall thresholds with email notifications.

Report Generation

Generate PDF and CSV reports with statistics and custom date ranges for compliance and analysis.

Multi-Station Support

Manage multiple weather stations from a single dashboard with user and organization assignments.

Dropbox Sync

Automatically sync TOA5 data files from Dropbox for seamless data collection from LoggerNet.

About Stratus

Stratus is a professional weather monitoring platform built for Campbell Scientific dataloggers. It provides a modern web interface for viewing real-time weather data, historical analysis, and report generation.

Supported dataloggers:

  • Campbell Scientific CR300
  • Campbell Scientific CR1000X
  • Campbell Scientific CR6
  • Other dataloggers using TOA5 format

Data collection methods:

  • Dropbox Sync - Automatic file synchronization
  • HTTP POST - Station pushes data to server
  • TCP/IP - Direct Ethernet or WiFi connection
  • LoRa Radio - Long-range wireless
  • 4G/LTE Cellular - Mobile data connection
